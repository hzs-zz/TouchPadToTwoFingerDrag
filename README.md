你是否曾遇到出门在外要打开电脑画图纸但是却没有带鼠标的尴尬局面😢

别急，此项目诞生就是为了解决这样的问题，触摸板也可以用来愉快的画图😁

由blender的触摸板适配获取灵感，触摸板的双指移动模拟鼠标中键的平移与缩放
按下alt+shift+m即可控制开启关闭

本人才开始入门c#，对于xaml，gui相关一窍不通，所以没有图形化界面。。。
